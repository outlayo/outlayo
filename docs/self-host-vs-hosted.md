# Self-Host vs Hosted

Use this as the baseline expectation matrix.

| Area | Self-Host | Hosted |
|---|---|---|
| Runtime deployment | You deploy and operate | Managed by Outlayo |
| Secrets management | You own keys and rotation | Managed by Outlayo |
| Upgrades and patching | You own rollout timing | Managed rollout |
| Backups and DR | You own backup/disaster recovery | Managed by Outlayo |
| OSS code access | Full source available | Built on same OSS foundation |

## Guideline

Product behavior should remain transparent and contributor-friendly, while hosted operational controls remain private for safety and reliability.
